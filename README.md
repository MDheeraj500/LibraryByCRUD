# Library-CRUD

* A Library Management System using JS-Node,express.
* DataBase used: MongoDB.
* MongoDB Collection name - 'Books'.
* Front-end done using AdminLTE template.

## Command to run first:-
```
$ node server.js
```
* open localhost:3000 on your browser.
